# Remove Logo Title Background

## Goal

On the storefront landing page, collection pages, and category pages, remove the background styling behind the title text that sits next to the logo.

## Scope

Apply to:

- Landing page header/logo title area
- Collection page header/logo title area
- Category page header/logo title area

## Expected UI

- Keep the logo visible.
- Keep the title text next to the logo.
- Remove the title background block/pill/card/badge effect.
- The title should render as plain text beside the logo.
- Preserve spacing, alignment, responsive behavior, and clickable/navigation behavior if present.

## Notes

- Do not remove the page title itself.
- Do not change backend APIs.
- If the same header/logo component is shared across these pages, update the shared component once.
- If each page has separate markup/styles, update each page consistently.
