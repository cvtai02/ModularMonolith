_All command are supposed to run in root folder_  
_Prerequisite: run `dotnet tool restore & npm install` beforehand_ 
---------------------------------------------------------------------------------------------


### Add migration
```bash
.\devtools\ef-add.ps1 -Module Identity -Name AddUserTable
```
---------------------------------------------------------------------------------------------

### Update database (latest or specific)
```bash
.\devtools\ef-update.ps1 -Module ProductCatalog
.\devtools\ef-update.ps1 -Module ProductCatalog -Name SomeMigration
```
---------------------------------------------------------------------------------------------

### Reset migration (rollback + remove last migration)
```bash
.\devtools\ef-reset.ps1 -Module Content -PreviousMigration InitialCreate
```
---------------------------------------------------------------------------------------------

### Api types generate (Require API is running)
_Don't use this command if run api (F5) in vscode. it's integrated in laucnh process_
```bash
.\devtools\openapi-typescipt.ps1
```
